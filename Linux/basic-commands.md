# Linux Basic Commands

## Objective

Learn and practice basic Linux commands used in daily terminal operations.

---

## 1. pwd

**Full Name:** Print Working Directory

**Function:**
Displays the current working directory.

**Example:**

```bash
pwd
```

---

## 2. ls

**Full Name:** List

**Function:**
Displays files and directories in the current location.

**Example:**

```bash
ls
```

---

## 3. cd

**Full Name:** Change Directory

**Function:**
Moves between directories.

**Example:**

```bash
cd Linux
```

Return to the previous directory:

```bash
cd ..
```

---

## 4. mkdir

**Full Name:** Make Directory

**Function:**
Creates a new directory.

**Example:**

```bash
mkdir Practice
```

---

## 5. touch

**Function:**
Creates a new empty file.

**Example:**

```bash
touch notes.txt
```

---

## Practice

Commands executed:

```bash
pwd
ls
mkdir Practice
cd Practice
touch notes.txt
ls
cd ..
```

Result:

Successfully created a directory, navigated into it, created a file, and listed its contents.
