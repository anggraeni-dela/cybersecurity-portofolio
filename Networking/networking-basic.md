# Networking Basics

## Objective

Understand the basic concepts of computer networking that are essential for cybersecurity.

---

## What is a Network?

A network is a group of devices connected together to communicate and exchange data.

---

## IP Address

An IP address is a unique identifier assigned to a device on a network.

Example:

192.168.1.10

---

## Port

A port is a communication endpoint used by network services.

Common ports:

| Port | Service |
|------|----------|
| 22 | SSH |
| 53 | DNS |
| 80 | HTTP |
| 443 | HTTPS |

---

## Protocol

A protocol defines how devices communicate.

Examples:

- TCP
- UDP
- HTTP
- HTTPS

---

## Practice

Commands executed:

```bash
ip a
hostname -I
ping 8.8.8.8 -c 4
```

---

## Result

Successfully displayed network interface information, local IP address, and verified internet connectivity.

---

## Conclusion

Understanding IP addresses, ports, and protocols is fundamental before using network security tools such as Nmap and Wireshark.
